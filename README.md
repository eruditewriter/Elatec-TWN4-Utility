# Elatec-TWN4-Utility
Python program that provides for reading and writing of access cards using the Elatec TWN4 with simple protocol.<br>
It provides:<br>
&emsp;*Reading UID for any card<br>
&emsp;*MIFARE Classic read and writes<br>
&emsp;*Allegion DESFire reads<br>
&emsp;*PKOC<br>
&emsp;*Send custom commands<br>
<br>
Uses pyCryptodome and ecdsa<br>
&emsp;'pip install pycryptodome'<br>
&emsp;'pip install ecdsa'<br>
