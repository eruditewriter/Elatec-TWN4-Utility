# Elatec-TWN4-Utility
Python program that provides for reading and writing of access cards using the Elatec TWN4 with simple protocol.
It provides:
    *Reading UID for any card
    *MIFARE Classic read and writes
    *Allegion DESFire reads
    *PKOC
    *Send custom commands

Uses pyCryptodome and ecdsa

'pip install pycryptodome'
'pip install ecdsa'
